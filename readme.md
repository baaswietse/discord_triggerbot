# Discord triggerbot
This bot takes track of how much people get triggered

## Commands
> **trigger** *@user: triggers a user*
> **triggerstats**: gets the triggered stats of all user*
> **triggerstats** *@user: gets the triggered stats of a user*
> **anime tidies**: oooohn
> **gay**: No u!
